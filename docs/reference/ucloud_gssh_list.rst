ucloud gssh list
----------------

List all GlobalSSH instances

Synopsis
~~~~~~~~


List all GlobalSSH instances

::

  ucloud gssh list [flags]

Examples
~~~~~~~~

::

  ucloud gssh list

Options
~~~~~~~

::

  --region     string       Optional. Assign region (default "cn-bj2") 

  --project-id     string   Optional. Assign project-id (default "org-ryrmms") 

  --help, -h                help for list
